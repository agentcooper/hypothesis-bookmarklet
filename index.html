How it works

When you run the bookmarklet it will:

  1. Attempt to save the current selection (if it exists) to Hypothesis
  2. Fetch existing annotations for the current page

Created annotations are public by default.

You can press on the highlight to set a note.

To remove the highlight, set the note to "x".
